# FEISHU-table-script-pdf2md
Script for FEISHU parse PDF to Markdown.

## How to start

1. Copy `.env.sample` to the same path and rename to '.env'. Complete all configurations in `.env`.

```shell
cp .env.sample .env
```

2. Install all packages with `pip`

```shell
python -m pip install -r requirements.txt
```

3. Just run `index.py`

```shell
python index.py
```
